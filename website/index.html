GIGUE

Get Version

0.0.1

* Description

‘GIGUE’ is a program for structure-sequence homology detection using environment-specific substitution tables and structure-dependent gap penalties.

* Features

  • Environment-specific substitution tables and structure-dependent gap penalties
  • Sequence-sequence, profile-sequence, and profile-profile alignment algorithms
  • Dynamic selection of alignment algorithms (Global/Glocal)
  • Ruby APIs (doc)

* Requirements

  • ruby 1.9.1 or above (http://www.ruby-lang.org)
  • rubygems 1.3.4 or above (http://rubygems.org/)

Following RubyGems will be automatically installed if you have rubygems installed on your machine

  • bio
  • narray
  • parallel
  • RubyInline

* Installation

~user $ gem install gigue-<VERSION>.gem

* Basic Usage

To build a profile,
~user $ gigue build -t template.tem -e essts.mat -o profile.gig
To search against a sequence database using the profile,
~user $ gigue search -p profile.gig -s sequences.fa -o hits.txt
To align a sequence to the profile,
~user $ gigue align -p profile.gig -s sequence.fa -o align.pir

* Options

GIGUE commands are:
    build     build profile(s) from JOY template(s) using ESSTs
    search    search query sequence(s) or alignment against target profile(s)
    align     align query sequence(s) or alignment with target profile(s)

- gigue build: building a structural profile

Usage: gigue build [options]

Options:
    -t, --template FILENAME          set JOY template file
    -e, --essts FILENAME             set environment-specific substition tables file
    -w, --weighting INTEGER          set weighting scheme
                                     0    EqualWeighting  -- weighting each sequence equally
                                     1    BlosumWeighting -- weighting scheme based on single linkage clustering
                                     2    VAWeighting     -- Vingron and Argos weighting (default)
    -o, --output FILENAME            set an output profile file (default: STDOUT)
    -v, --verbose INTEGER            show detailed console output
                                     0    Error level (default)
                                     1    Warning level
                                     2    Information level
                                     3    Debugging level
    -h, --help                       show this help message

- gigue search: searching a sequence database using a structural profile

Usage: gigue search [options]

Options:
    -p, --profile FILENAME           set target profile
    -s, --sequence FILENAME          set query sequence(s)
    -o, --output FILENAME            set output file name (default: STDOUT)
    -t, --toprank INTEGER            output scoring information about top N HITs
    -z, --zcutoff FLOAT              output scoring information about HITs with Z-scores > cutoff
    -c, --process INTEGER            set number of processes to use (default: 1)
    -a, --algorithm INTEGER          set alignment algorithm
                                     0      Global/Glocal (default)
                                     2      Local
    -v, --verbose INTEGER            show detailed console output
                                     0      Error level (default)
                                     1      Warning level
                                     2      Information level
                                     3      Debugging level
    -h, --help                       show this help message

- gigue align: aligning a sequence to a structural profile

Usage: gigue align [options]

Options:
    -p, --profile FILENAME           set target profile
    -s, --sequence FILENAME          set query sequence(s)
        --alignment FILENAME         set query alignment
    -o, --output FILENAME            set output file name (default: STDOUT)
    -a, --algorithm INTEGER          set alignment algorithm
                                     0      Global/Glocal (default)
                                     2      Local
    -v, --verbose INTEGER            show detailed console output
                                     0      Error level (default)
                                     1      Warning level
                                     2      Information level
                                     3      Debugging level
    -h, --help                       show this help message
